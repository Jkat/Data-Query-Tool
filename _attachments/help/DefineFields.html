Help: Define Fields

To define fields for a query, begin by selecting an instrument or category of data fields (e.g. demographics) from the "Choose category" dropdown select box. All field names and field descriptions from that category will appear in a table below. The number of fields displayed per page can be modified in the top left corner of this table. Any results exceeding this display limit will be displayed on subsequent pages within the table, accessible via buttons labelled First, Previous, Next and Last, visible in the lower right corner after scrolling to the bottom of the field display table.

To add a field to your query, click anywhere in the field row to highlight the field and add it to the list of "Selected Fields" at bottom.
To remove a selected field from a query, click in the highlighted field row in the list of all fields, or click on its "Remove" button in the Selected Fields table.

If all fields from a particular instrument or category should be included in the query, use the "Add all" button next to the "Choose category" dropdown to select all fields. Click "Remove all" to remove all fields in this category from the query.

Note that demographic fields such as IDs, gender, site and cohort are typically desirable to include in most queries. To add these fields, select "demographics" from the "Choose category" dropdown to view all available demographic fields, and then click to add specific fields.

Once all desired fields are listed under "Selected Fields", continue to the "Define Filters" tab.
